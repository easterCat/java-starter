GitHub Pages

GitHub Pages 是 GitHub 提供的静态网站托管服务。https://<username>.github.io/<repository>/

GitHub Pages 的静态资源支持下面 3 个来源：

  • master 分支
  • master 分支的 /docs 目录
  • gh-pages 分支
